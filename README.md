Real-Time Collaborative Drawing Canvas

A web-based collaborative drawing application built with **Node.js**, **Socket.IO**, and the **HTML5 Canvas API**, allowing multiple users to draw on a shared canvas in real time.  
It supports smooth synchronization, undo/redo operations, and an intuitive creative UI for live collaboration and visualization.

---

 Features

-  **Real-Time Collaboration** — Multiple users can draw together instantly.
-  **Smooth Drawing Experience** — Uses HTML5 Canvas with continuous path tracking.
-  **Undo / Redo** — Global operation log with server-assigned sequence IDs.
-  **Persistent Storage (Optional)** — Periodic snapshots and operation logs can be saved to MongoDB Atlas.
-  **Server Authoritative Model** — Operations validated and sequenced by the backend.
-  **Creative, Modern UI** — Simple and colorful layout for easy interaction.
-  **Live Sync via Socket.IO** — Robust reconnection and room-based sessions.
